# Tic-Tac-Toe

This is a simple Tic-Tac-Toe game built using HTML, CSS, and JavaScript.
It lets two players play against each other on the same device.
The UI is clean, responsive, and easy to interact with.

## Features

- Two-player mode 
- Shows win, draw, and reset states
- Simple and clean UI
- Responsive layout for mobile and desktop

## Tech Stack

- **HTML**
- **CSS**
- **JavaScript**

## How to Run

Download or clone the project:

```bash
git clone https://github.com/jvkedev/tic-tac-toe-app.git
```

Open index.html in your browser.

## Notes

- This is a static frontend project.
- No backend, database, authentication or API calls are used.
- Perfect for beginners practicing DOM manipulation and game logic.
